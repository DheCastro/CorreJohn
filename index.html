Correjohn

Game Mobile desenvolvido com Framework CoronaSDK

View project on GitHub

Descrição:

Corre John é um jogo side-scrooling que tem como personagem principal o John. O jogador terá que tocar nos alimentos não saudáveis evitando-o que cheguem até o John e assim estará ajudando o John a alcançar sua meta que é chegar aos 50kg.

Regras e elementos importantes do Jogo:

O Objetivo é fazer com que John alcance os 50kg. A cada 10 ou 20 quilos a menos vai aumentando a produção de mais alimentos na tela. Cada alimento saudável que colide com John é 1kg a menos e cada alimento não saudável é 1kg a mais. O jogador perde caso o peso do John chegue a 200kg.

Tecnologia:

Foi utilizado somente o framework Corona SDK para o desenvolvimento do jogo. O maior desafio foi aprender a linguagem e dominar algumas técnicas em um período curto e também trabalhar com edição de imagens para que o jogo tivesse um layout amigável com o usuário.

Referências:

O jogo foi inspirado no game mobile zombie tsunami, porém já existem diversas versões inspiradas nesse game no mercado.
